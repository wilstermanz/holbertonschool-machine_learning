## About The Project

This repository is broken up into [Math](https://github.com/wilstermanz/holbertonschool-machine_learning/tree/main/math), [Supervised Learning](https://github.com/wilstermanz/holbertonschool-machine_learning/tree/main/supervised_learning), and [Unsupervised Learning](https://github.com/wilstermanz/holbertonschool-machine_learning/tree/main/unsupervised_learning) sections. Each contains projects relating to that specific discipline. Please see the README within each subrepository for more infomation on the projects that it contains.

This reposistory can be cloned locally. Test files for each task are usually included as _task_number_-main.py

Jupyter notebooks are included to follow along for some projects.
